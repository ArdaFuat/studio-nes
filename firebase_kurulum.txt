FIREBASE KURULUMU - NESS ART CANLI ADMIN (STORAGE YOK)
=====================================================

Bu sürümde Firebase Storage kullanılmaz. Ürün bilgileri Firestore'a kaydedilir.
Görseller için admin panelde sadece görsel yolu / URL girilir.

1) Firebase projesi
-------------------
Firebase Console'da proje hazır: ness-art

2) Config
---------
assets/js/firebase-config.js içinde Firebase config bilgileri ve admin mail listesi hazırdır.
Admin mailleri:
- ardamevk12@gmail.com
- ardabesiktasarda@gmail.com
- nesibeyakann@gmail.com

3) Authentication
-----------------
Firebase Console > Authentication > Sign-in method
Email/Password yöntemini aktif et.

Authentication > Users kısmından şu kullanıcıları oluştur:
- ardamevk12@gmail.com
- ardabesiktasarda@gmail.com
- nesibeyakann@gmail.com

Netlify domaini için:
Authentication > Settings > Authorized domains içine ness-art.netlify.app ekli olsun.

4) Firestore Database
---------------------
Firebase Console > Firestore Database > Rules bölümüne aşağıdaki kuralları koy ve Publish/Yayınla de:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    function isAdmin() {
      return request.auth != null && request.auth.token.email in [
        "ardamevk12@gmail.com",
        "ardabesiktasarda@gmail.com",
        "nesibeyakann@gmail.com"
      ];
    }

    match /artworks/{artworkId} {
      allow read: if true;
      allow create, update, delete: if isAdmin();
    }
  }
}

5) Storage gerekli değil
------------------------
Bu pakette Firebase Storage scripti ve dosya yükleme kaldırıldı.
Storage açmana, Blaze'a geçmene veya kart eklemene gerek yok.

6) Görsel ekleme mantığı
------------------------
Admin paneldeki "Görsel yolu / URL" alanına iki tür değer yazabilirsin:

A) Site içine koyduğun görsel:
assets/img/white_lilies.jpeg
assets/img/yeni-tablo.jpg

B) Dışarıdaki direkt görsel URL'si:
https://site.com/gorsel.jpg
https://site.com/gorsel.png
https://site.com/gorsel.webp

Önemli: Google Drive/Instagram sayfa linki çoğu zaman direkt görsel linki değildir. En sağlam yöntem yeni görselleri assets/img klasörüne koyup Netlify'a güncel klasörü tekrar yüklemektir.

7) Netlify'a yükle
------------------
Zip'i aç. İçindeki ness_art_site klasörünün içeriğini Netlify'a yükle.
Admin panel:
https://ness-art.netlify.app/admin.html

8) İlk ürünleri aktar
---------------------
Admin paneline giriş yap.
"İlk Ürünleri Aktar" butonuna bir kez bas.
Bundan sonra ürün ekleme/düzenleme/silme işlemleri Firestore üzerinden canlı çalışır.

9) Günlük kullanım
------------------
- Ürün ekle: formu doldur, görsel yolu/URL yaz, Kaydet.
- Ürün düzenle: sağdaki listeden Düzenle.
- Ürün sil: Sil.
- Sırayı değiştir: Yukarı / Aşağı.
- Yeni görsel için en pratik yol: görseli assets/img klasörüne koy, admin panelde assets/img/dosya-adi.jpg yaz.


STORAGE GEREKMEZ:
Bu pakette Firebase Storage kullanılmaz. Sadece Authentication + Firestore yeterlidir. Görseller admin panelde seçildiğinde küçültülerek Firestore içindeki image alanına kaydedilir.
